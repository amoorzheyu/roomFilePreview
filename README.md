# roomFilePreview

多人同步预览（PDF/Markdown）的小工具：创建/进入房间，房主上传并控制滚动，观众实时同步。

## 开发启动

1. 安装依赖

```bash
pnpm install
```

2. 启动（前后端同时）

```bash
pnpm run dev
```

默认：
- 前端：`http://localhost:5173`
- 后端：`http://localhost:8787`
